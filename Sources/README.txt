Pour compiler :
- tous d'un coup : exécuter le makeAll.bat
- un à la fois : aller dans le dossier correspondant et exécuter le make.bat

Cela vous produira des fichiers *Main.exe dans chaque dossier.

Vous pouvez modifier à votre guise les fichiers : garder juste la même signature pour les fichiers *Game et *Strategy pour rester compatibles entre vous.

Les algorithmes minimax et alpha-beta sont à coder dans les fichiers *Strategy.ml. Il est possible qu'il faille modifier certains .mli et éventuellement les .bat si vous
rajouter d'autres fichiers.

Pour l'exercice d'apprentissage sur le Morpion, le fichier à modifier est MorpionLearning.ml.
